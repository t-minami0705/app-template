# 🛠️Class : Const
#概要 
Pythonで定数を安全委管理するためのクラス。
Pythonには言語使用として定数が存在しないため、本クラスでは一度定義した属性への再代入を禁止し、定数の上書きを防止する仕組みを提供する。
モジュールの読み込み時にモジュール自信を_Constクラスのインスタンスへ書き換えることで、通常のモジュールと同じ感覚で定数を定義・参照できる。

#主な特徴
- Pythonで定数を疑似的に実現する。
- 一度設定した値の変更を禁止する。
- 同じ値の再代入は許可される。
- 異なる値への再代入時は"\_conostTYpeError"を送出する。
- モジュールをインスタンスへ置き換えることで、モジュール属性として定数を管理できる。

#使用イメージ
``` Python
import Const

Const.MAX_COUNT = 100      # 定義
Const.MAX_COUNT = 100      # OK（同じ値）
Const.MAX_COUNT = 200      # NG（例外発生）
```

#実装のポイント
- "\_\_setattr\_\_()"をオーバーライドし、属性設定時に既存値との比較を行う。
- "sys.modules[\_\_name\_\_] = \_Const()"により、モジュールオブジェクトを"\_Const"インスタンスへ置き換える。
- 利用者クラスを意識するkとはなく、通常モジュールのように定数を利用できる。

>[!Note] ポイント
>これは、Python でよく利用されるモジュールをシングルトンオブジェクトとして扱うテクニックの１つである。
